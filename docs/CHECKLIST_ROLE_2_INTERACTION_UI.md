# Webster: чеклист роли 2 (интерактив и интерфейс)

Цель роли: реализовать UX слоя редактора, взаимодействия с объектами и панели управления.

## 1. Каркас интерфейса редактора

- [ ] Реализовать layout редактора: toolbar, sidebar, canvas area, footer.
- [ ] Реализовать адаптивное поведение для laptop/tablet.
- [ ] Реализовать state панели инструментов.
- [ ] Реализовать state панели свойств выделенного объекта.
- [ ] Реализовать унифицированные UI-компоненты контролов.

## 2. Интерактивные события canvas

- [ ] Реализовать onMouseDown с разделением режимов (select/draw/text).
- [ ] Реализовать onMouseMove для drag/resize/rotate.
- [ ] Реализовать onMouseUp с фиксацией команды в history.
- [ ] Реализовать onWheel для zoom.
- [ ] Реализовать keyboard handlers (Delete, Ctrl+Z, Ctrl+Y, arrows).
- [ ] Реализовать модификаторы Shift/Ctrl для мультивыбора.

## 3. Select/Drag/Transform UX

- [ ] Реализовать визуальное выделение объекта.
- [ ] Реализовать selection box для мультивыбора.
- [ ] Реализовать drag-and-drop объектов.
- [ ] Реализовать resize handles.
- [ ] Реализовать rotate handle.
- [ ] Реализовать snap к сетке (если включено).
- [ ] Реализовать ограничение перемещения в пределах canvas (опционально).

- [ ] Реализовать bold/italic/underline для текста.
- [ ] Реализовать настройку толщины линии.
